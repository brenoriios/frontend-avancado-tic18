Ideia do Aplicativo

Este aplicativo tem como função gerenciar os usuários cadastrados no sistema, bem como associar arquivos a esses usuários para que eles possam visualizar quando se conectarem com suas contas.

O cadastro de novos usuários precisa ser feito por um usuário administrador através do painel de administrador.

Uma vez cadastrado o novo usuário receberá um email solicitando redefinição de senha e, após redefiní-la, poderá acessar o aplicativo e ver os arquivos que foram associados a ele.

Visão geral do aplicativo

O aplicativo conta com um menu lateral esquerdo com os seguintes itens:

Área do administrador:

Usuários -> Lista todos os usuários comuns cadastrados no sistema

Área do usuário:

Perfil -> Exibe o perfil do usuário

Arquivos -> Lista todos os arquivos associados ao usuário

Outros:

Tutorial -> Este guia mostrando como usar o aplicativo

Conta:

Sair -> Desconecta da conta e encerra a sessão

Visão da lista de usuários

Na tela de usuários, o usuário pode clicar no botão de excluir para excluir o usuário do sistema, clicar no botão de editar para editar o cadastro do usuário, ou o botão de adicionar arquivo para associar um arquivo ao usuário.

É possível ainda cadastrar novos usuários ou filtrar os usuários pelo nome.

Ao clicar no nome do usuário o sistema exibirá a tela de detalhes, com o perfil do usuário selecionado e os arquivos associados a ele.

Ao clicar no nome exibido em "Último Arquivo", o sistema abrirá o arquivo no navegador.

Tela detalhes do usuário

A tela de detalhes do usuário exibe os dados do usuário e os arquivos associados ao usuário.

É possível clicar no botão de excluir para excluir um arquivo do usuário.
